Legendre Symbol

1. Legendre Symbol

The Legendre symbol, denoted \((a/p)\) is a function of two integers describing Quadratic Residues. \(a\) is an integer, \(p\) is an odd prime. If \((a/p) = 1\) then \(a\) is a quadratic residue mod \(p\). If \((a/p) = -1\), then \(a\) is a quadratic non-residue. \((a/p) = 0\) for \(a = 0\).

See also: Properties of Legendre Symbol
