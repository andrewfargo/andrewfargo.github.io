Desc: A short update: I've added RSS support for my website!
Category: meta

Hi all, just a short update tonight.

I've spent the better part of the week trying to repair my old Singer machine (soon to be a post, hopefully). In the meantime, I've added RSS support for this blog! If you go to either my main page or my blog home, a graphical link to the RSS feed is now present: .

I was hoping to add support for an RSS subdomain, e.g. rss.dfargo.com (invalid link), but had trouble getting it to work with GitHub pages. Alas.

Next I think I want to get rid of some friction with uploading posts. Currently, I have to

  1. copy over a template HTML file,
  2. write the article,
  3. fill out the metadata in the article text
  4. fill out the metadata in the HTML
  5. fill out the metadata in the RSS feed
  6. update the blog home;
and, this is all without support for categories, for which I would also have to update regularly. My ideal solution would be converting a LaTeX document into the proper HTML and RSS, as well as a tool to convert RSS to a set of navigational pages. This is in the works and I will create a blog post should I finish it.

That's all, goodnight!
